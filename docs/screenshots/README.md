## Generating Screenshots

### Using Screenshot Tools

**macOS:**
```bash
# Use built-in screenshot tool (Cmd+Shift+4)
# Or use screencapture command
screencapture -T 0 -x screenshot.png
```

**Windows:**
- Use Snipping Tool or Windows+Shift+S
- Or use ShareX for advanced screenshots

**Linux:**
```bash
# Using scrot
scrot screenshot.png

# Using gnome-screenshot
gnome-screenshot -f screenshot.png
```

## Image Requirements

- **Format:** PNG for screenshots, GIF/MP4 for demos
- **Resolution:** 1920x1080 (minimum)

- **File Size:** Screenshots < 2MB each, GIF < 5MB, Video < 50MB
